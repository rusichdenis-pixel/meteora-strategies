# Meteora DLMM High-Risk Strategies

Интерактивный дашборд стратегий для предоставления ликвидности на Meteora DLMM (Solana).

## 🎯 Стратегии
- ⚡ Momentum Sniper — снайпер по новым мемкоинам
- 🔄 Volume Harvester — сбор комиссий с активных пулов
- 🎯 Blue-Chip Meme — стабильный доход на устоявшихся токенах

## 🛠 Технологии
- React 18 + TypeScript
- Tailwind CSS
- Полностью адаптивный дизайн
